# Learn React DEMO

## List

 - React for question : `QuestionApp`
 - Tab : `TabApp`

## Use
 1. `npm install -g yo `
 2. `npm install -g generator-react-webpack`
 3. `npm install`
 4. `grunt serve`
 5. Chrome open `http://localhost:8000/webpack-dev-server/`
